# 1Pacent Product Brief

1Pacent is an AI-enabled property maintenance orchestration platform.

It connects:
- Tenants
- Landlords
- Property managers
- Tradies
- Platform administrators

The platform manages rental property maintenance requests, landlord approvals, tradie job allocation, tenant scheduling, job completion evidence, compliance work, and invoice/payment status.

It is not just a tradie marketplace.

It is a maintenance operating system for rental property management.

## Core Value Proposition

For property managers:
Reduce admin, improve compliance, automate approvals, and track maintenance end-to-end.

For tenants:
Report issues easily, upload evidence, choose access windows, and track progress.

For landlords:
Approve spend, view maintenance history, and keep properties compliant.

For tradies:
Receive nearby, pre-scoped, approved jobs and get faster completion confirmation.

## MVP Goal

Build a working Flutter web app that demonstrates the end-to-end workflow using mock data.
